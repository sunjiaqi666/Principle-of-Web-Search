### 北京邮电大学“网络搜索原理”课程(2019)
- 课程名称：网络搜索原理
- 教师：陈光 ([@爱可可-爱生活](https://weibo.com/fly51fly))
- 时间：周三上午 8:00-9:35
- 地点：教三楼239

本课程旨在通过实战的方式，理解搜索引擎背后的基本技术原理，并将其应用于解决实际问题。课堂将重点讨论“为什么”，而不仅仅是“怎么做”，希望能对同学们有所启发。
